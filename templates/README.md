# Templates

Reusable HTML templates for this site.

## Files
- `article-template.html` - for normal article pages
- `faq-template.html` - for Q&A / FAQ pages
- `category-index-template.html` - for category top pages

## How to use
1. Copy the closest template into `document/<category>/...html`
2. Replace placeholders like `{{PAGE_TITLE}}`
3. Keep the existing CSS path structure consistent with the target folder
